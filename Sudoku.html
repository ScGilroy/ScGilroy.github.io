I have found Classic Sudoku puzzles to be very interesting and a great logic puzzle. Recently I have come across multiple different variants which I have developed a solver to solve these variants in multiple different programming languages.

The Challenge:
How to develop a solver to solve different Sudoku variants within a reasonable time? How to determine all vaild solutions to the puzzle?

The Result:
A recursive solver which finds all possible solutions for the variant which is enabled. This solver is written in MatLab, Python and C++.

Logic Heavy Approach

To begin the project, I developed an algorithm for the classic sudoku puzzles in Matlab. This algorithm relies heavily on logic to place all values while iterating through the puzzle's 81 tiles. The algorithm can be broken down into two parts. The first part would determine all possible values for a specific tile (x,y). If there is only a single solution to the tile, it would place the answer into the box. The second algorithm determines if a row, column or box only has one missing value then places the value into the tile. The combination of these algorithms allows for the solver to solve classic sudoku puzzles between the difficulties of easy and hard.

Recursive Approach

The recursive approach utilizes a back-tracking methodology. The algorithm checks the constraints for the box, column and row for possible values that a tile can take then picks the first value. The algorithm continues moving through the 81 tiles. If the algorithm gets stuck, the program will backtrack and try the next possible value. This method allowed for the code to solve any level of Sudoku puzzle in very quick times. Additionally, the program will also output all possible solutions allowing the user to determine if a unique solution exists for the puzzle.

Additional Variants

Anti-King's Move Sudoku

Anti-King's Move Sudoku infuses chess and sudoku. This variant adds an additional constraint where all the digits (1-9) can not be a king's move away from each other. With all the classic sudoku constraints, the only tiles added to the constraint are the ones which are a single diagonal tile away.

Anti-Queen's Move Sudoku

Anti-Queen's Move Sudoku is very similar to Anti-King's Move but adds the diagonal constraint for all tiles diagonal to the tile in question. This constraint is only active on a single digit (usually on the value 9). If this constraint was active for all values, the puzzle would be infeasible and therefore no solution would be found.

Anti-Knight's Move Sudoku

Anti-Knight's Move Sudoku is adds the constraint of a Knight in chess, where the knight moves in an L shape. This constraint is active for all numbers and provides one of my favorite variants of Sudoku.

Format for the Chess Variants.
Orange - Anti-Queen | Red - Anti-King | Green - Anti-Knight

Results

A robust Sudoku solver for the Classic, Anti-King's Move, Anti-Queen's move, and Anti-Knight's move variants which runs within a reasonable time. These were tested on multiple puzzles and the outputs were correct for each programming language.
